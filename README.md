🚀 NiFi Data Transformation: Native Processors vs Custom Python Processor
📌 Overview

Project ini bertujuan untuk membandingkan dua pendekatan dalam melakukan data transformation di Apache NiFi:

1. Native NiFi Processors (Multi-Step Pipeline)
2. Custom Python Processor (Single-Step Processing)

Dataset diambil dari API:
👉 https://dummyjson.com/products

🎯 Objectives
Mengambil data dari API (InvokeHTTP)
Melakukan filtering dan transformasi data
Membandingkan kompleksitas antara:
- Banyak processor (native NiFi)
- 1 custom processor (Python)
